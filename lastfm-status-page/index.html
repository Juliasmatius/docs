Last.fm status page

Juli The Furry

This is a status page which shows what song a specific user is currently listening to. It displays the song name, artist, album and album art. In the page title it shows the artist and song name. Available in both dark and light mode and with and without buttons.

 

Using

  1. Download and install python3.
  2. Install flask and requests using
    pip3 install flask requests
  3. Open config.ini and configure the settings
    [config]

; The user which you want to track
LASTFM_USER=Example_user

; Api key from https://www.last.fm/login?next=/api/account/create (doesnt need to be the same user as tracked)
API_KEY = 11ea120a0ec0f68561ae3dbbc87f81a3

; Don't worry about this one. I don't even remember if it does anything
MAIN_PAGE_PATH = /
; Set the port where server is hosted
PORT= 80
; Keep blank unless you want to set flasks debug mode true
DEBUG=

[DEV]
; A standard user doesn't need to touch there. Only for development purposes
; Root of the api where data is collected from.
api_root=http://ws.audioscrobbler.com/2.0
; How often the scripts requests data from the api
REFRESH_TIME=15
; How often should the website refresh it's data.
WEBSERVER_REFRESH=15
  4. Run the script using
    python3 main.py
  5. Visit the website on 127.0.0.0.1:<port from config> (ps. the server is bound to 0.0.0.0 so you can use your hosts IP too)

Embedding

The webserver provides a /embed which can be used for embedded or display purposes. Adding an argument like ?hide_buttons to hide the pages buttons and ?dark_mode for getting dark mode.

# Just darkmode
/embed?dark_mode

# Just hiding buttons
/embed?hide_buttons

# Both
?embed?dark_mode&hide_buttons
